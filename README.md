#publier

A simple CMS written in Symfony2 using Sqlite.


## Installation

1. cd to web root
2. git clone http://github.com/aletzo/publier.git publier
3. chmod -R 777 app/cache app/logs



## How to clear cache

1. rm -rf app/cache app/logs
2. mkdir app/cache app/logs
3. chmod -R 777 app/cache app/logs
